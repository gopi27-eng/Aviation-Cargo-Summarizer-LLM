# Eizen AI - NLP Assignment
**Candidate:** Gopi Borra
**Task:** Option D - Natural Language Processing (LLM)

### Project Overview
I implemented a Zero-Shot Inference summarization tool using the `google/flan-t5-base` model. Leveraging my 5-year background in Aviation Security at Quikjet Cargo Airlines, I used a real-world cargo screening scenario as the input data to test the model's summarization capabilities.

### Technical Implementation
- **Vibe Coding Approach:** Developed using Cursor AI to rapidly prototype the solution.
- **Model:** FLAN-T5 (Base)
- **Method:** Zero-Shot Inference with an explicit prompt template (`summarize: `).
- **Environment:** Local Python 3.13 with Hugging Face Transformers.

### How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python app.py`
